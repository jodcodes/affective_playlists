## Why
The repository now has OpenSpec installed, but there is no initial change artifact set to guide future spec-first work.

## What Changes
- Add a baseline OpenSpec change package for this repository.
- Define a simple workflow standard for proposing and applying changes.
- Document how this Python project should structure OpenSpec change requests.

## Impact
- Contributors can start from an existing pattern instead of a blank folder.
- AI-assisted implementation can follow explicit artifacts (proposal, design, tasks, specs).
- Lower onboarding friction for future maintenance and feature work.
